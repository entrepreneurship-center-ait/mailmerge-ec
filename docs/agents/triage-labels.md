# Triage Labels

Label mapping for the five canonical triage roles:

| Role | Label |
|------|-------|
| needs evaluation | `needs-triage` |
| waiting on reporter | `needs-info` |
| AFK-ready, fully specified | `ready-for-agent` |
| needs human dev | `ready-for-human` |
| will not fix | `wontfix` |

Apply these exact strings when moving issues through the triage state machine.
